Welcome to the Intercom Chrome Extension!


To get started, you need to save your Intercom API credentials.

Your API credentials will be stored on your computer and the developers of this extension or any other parties will not have access to them. You can even check out the code of this application on Github here.

See step by step instructions to getting your Intercom API credentials

To get and save your credentials follow these steps:

  1. Sign into Intercom here
  2. Hover on the person icon at the bottom left of the screen and click on "App Settings"


  3. Click on the "API Keys" tab in the left side menu


  4. Click on "Create Read Only Access API Key"


  5. Click "Accept and Continue" if prompted
  6. Your new API is now created and should appear in the table above where you clicked to create
  7. Now copy the App ID and API Key into the input fields below and click save!
